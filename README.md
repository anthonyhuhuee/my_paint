## Run Locally

Clone the project

```bash
  git clone git@github.com:anthonyardhuin/my_paint.git
```

Go to the project directory

```bash
  cd my_paint
```

Compile the project

```bash
  make
```

Start the game

```bash
  ./my_paint
```
